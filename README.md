# HBCE Autonomous Systems Lab

Research and development environment for autonomous and semi-autonomous machines.

This lab develops a universal control and audit layer compatible with:

- drones
- ground robots
- humanoid robots
- industrial machines
- cyber-physical infrastructure

The objective is to create a hardware-independent behavioral core that can operate across different machines while remaining compliant with EU-first, audit-first and fail-closed operational principles.

## Core philosophy

Control is not inside the machine.  
Control is inside the decision layer.

Machines become interchangeable bodies.  
The behavioral core remains persistent.

## Current phase

Phase 1 — browser-based simulation  
Development of a deterministic control environment:

- follow / hover / explore-safe logic
- obstacle avoidance
- append-only event logging
- hash-chained audit trail
- hardware-independent architecture

Phase 2 — hardware connection  
Integration with real devices via SDK:

- drone SDK (Tello or equivalent)
- ground robots
- future humanoid systems

## Positioning

This repository represents an early R&D layer of the HBCE ecosystem focused on:

Autonomous systems  
Operational identity  
Machine behavior control  
Audit-first robotics  

Designed for industrial, research and EU-level applications.
